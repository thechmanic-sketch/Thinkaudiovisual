# Think Audio Visual

Stage, sound, lighting and AV production website for Think Audio Visual (Durban, South Africa).

## Pages
- `index.html` — Home
- `about.html` — About
- `services.html` — Services
- `portfolio.html` — Portfolio & Clients
- `gallery.html` — Gallery
- `contact.html` — Contact & Booking (WhatsApp-integrated)

## Stack
Static HTML/CSS/JS. Motion powered by GSAP + ScrollTrigger (loaded via CDN). Fonts: Poppins, Montserrat, Sora.

## Deploy
Push to `main` and enable GitHub Pages (Settings → Pages → Deploy from branch → `main` → `/root`).
